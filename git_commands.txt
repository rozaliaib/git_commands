git init - инициализируем git репозиторий

git status - проверка текущего состояния репозитория

git log - выводит список всех предшествующих коммитов

git add "file_name" - добавляет к отслеживанию выбранный файл или файлы( имена файлов пишутся через пробел)

git commit -m "some text" - сохраняем текущие отслеживаемые изменения в коммит, "some text" - комментарий к коммиту

git checkout - позволяет перемещаться между ветками

git branch - позволяет создавать, просматривать, переименовывать и удалять ветки
